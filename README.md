## Overview

This project is a retrieval-augmented generation assistant built for two corpora: Ghana presidential election results (CSV) and Ghana’s 2025 Budget Statement and Economic Policy (PDF). Retrieval, chunking, embedding, vector search, scoring, prompt construction, and logging are implemented in plain Python. End-to-end RAG frameworks (for example LangChain or LlamaIndex) are not used.

Data are fetched at runtime from the URLs defined in `rag_config.py` and cached under `data/`.

## Running locally

Prerequisites: Python 3.11 or newer recommended.

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
$env:OPENAI_API_KEY = "your-key-here"
streamlit run ca_10022200167.py
```

The API key may instead be entered in the application sidebar for the current session only.

On first run, Hugging Face models are downloaded and a FAISS index is built under `index_store/`. Expect several minutes on a first machine or cold cache.

## System summary

- **Ingestion and chunking** (`rag_data.py`): CSV rows are normalised and emitted as one text chunk per row. The PDF is extracted with `pypdf` and segmented with a sliding character window; window length and overlap are set in `rag_config.py`.
- **Retrieval** (`rag_retrieve.py`): dense vectors from Sentence Transformers; FAISS inner-product search on unit-normalised embeddings; optional lexical overlap in the fusion stage; cross-encoder re-ranking; lightweight query expansion; domain-aware weighting and optional session-side source boosting from the UI.
- **Prompting** (`rag_prompt.py`): context-only system instructions, greedy packing under a character budget, and selectable prompt variants for experiments.
- **Generation** (`rag_llm.py`): OpenAI Chat Completions API; a separate path calls the model without retrieved context for baseline comparison.
- **Interface and trace** (`ca_10022200167.py`): Streamlit UI; expandable views for retrieved passages, scores, assembled prompt, and optional baseline output; append-only trace in `logs/rag_pipeline.log`.
